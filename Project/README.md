# 🧠 MindGuard – Mental Health Tracking Dashboard

MindGuard is a frontend-based web application designed to help users track their mental well-being through journaling, mood analysis, and habit tracking.

This project focuses on building a clean, scalable, and interactive user interface that simulates mental health insights using user inputs.

---

## 🚀 Features

### 📊 Dashboard

* Mood timeline visualization
* Daily check-in (Great / Okay / Low / Bad)
* Habit builder (water, meditation, walking, gratitude)
* Smart suggestions based on user activity

### 📝 Risk Analysis (Simulated)

* Journal input system
* Sentiment-based UI feedback (simulated)
* Risk score visualization
* Behavioral inputs:

  * Sleep hours
  * Physical activity
  * Social interaction

### 🔍 Discover

* Self-discovery sections
* Emotional intelligence overview
* Stress pattern insights (UI-based)

### 🧘 Relax & Focus

* 4-7-8 breathing exercise UI
* Focus tap game UI
* Gratitude jar
* Zen garden concept

### 📜 History

* Journal logs
* Mood journey tracking
* Long-term overview stats

### 🆘 Resources

* Crisis support UI
* Coping mechanisms
* Therapy suggestions

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Routing:** React Router
* **Styling:** CSS (Dark Theme UI)
* **State Management:** useState, useEffect

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/mindguard.git
cd mindguard
npm install
npm run dev
```
